FTA Portal Application Programing Interfaces – Policy on use and dissemination

FTA Portal API

The FTA Portal Application Programming Interfaces (API) distribute information about preferential tariff and rules of origin relating to the trade in goods under Australia’s free trade agreements (FTAs).

Free unlimited access

Free unlimited access is available for all users of the FTA API. There is no limit on the number of queries, although this will be reviewed depending on usage and performance.

The tariff and rules of origin data within the FTA Portal, and as presented through the API, is free of charge, can be disseminated widely and has unlimited access. Usage of the API is closely monitored. The Commonwealth of Australia, represented by the Department of Foreign Affairs and Trade (DFAT), reserves the right to restrict or prohibit access, including on a permanent basis, to users who may engage in wilful misuse, contrary to the objectives of the project.

While DFAT aims to maintain high service availability, DFAT does not make guarantees nor accept any legal liability whatsoever arising from, or connected to, the performance quality of the API service. For further information regarding availability, please refer to the Amazon Web Services Service Level Agreement.

The sample code for this API is published under The MIT License.

Disclaimer

The use of the FTA API is subject to the disclaimer contained on the FTA Portal website at: https://ftaportal.dfat.gov.au/disclaimer.

In addition, users should note that the API is subject to change over time. Users are responsible for ensuring they are accessing the latest version. For example, HS codes are updated periodically. Updates will be communicated through the FTA Portal Developer Resources Blog, available at http://ausftas.github.io/.

Additional information on the FTAs

For further information on the FTAs, please visit https://ftaportal.dfat.gov.au/ or http://fta.gov.au.

Contact and feedback

DFAT welcomes feedback on your experience with using the FTA Portal and the FTA API, including suggestions to improve the service or to fix errors or other problems. DFAT is under no obligation to provide technical support for its use.

Post your feedback and questions about the API on the FTA Portal API GitHub issues page. Both the GitHub issues page and the API blog are maintained by National ICT Australia under contract to DFAT.

For feedback on the FTA Portal, please email FTAportalfeedback@dfat.gov.au.
