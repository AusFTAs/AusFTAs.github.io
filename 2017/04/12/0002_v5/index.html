Three new agreements now available

Three additional agreements were added to the site on 12 April 2017:

  • ASEAN-Australia-New Zealand Free Trade Agreement (AANZFTA)
  • Singapore-Australia Free Trade Agreement (SAFTA)
  • Thailand-Australia Free Trade Agreement (TAFTA)

The API has been updated accordingly. Please note that v1 API elements which were previously deprecated have been completely removed in this version, see http://ausftas.github.io/apidocs/. We continue to encourage feedback and questions on the API service here.
